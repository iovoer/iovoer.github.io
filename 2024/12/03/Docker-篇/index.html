本章着重介绍Docker的命令和实战演演练 介绍和理论部分较少—

docker-容器

容器是隔离的环境中运行的一个进程,如果进程结束,容器就会停止.

容器的隔离环境,拥有自己的ip地址,系统文件,主机名,进程管理,相当于一个mini的系统.

docker安装

本章系统为 Centos-7

由于使用到了端口映射需要打开系统内核转发功能

docker-images 管理

docker search 搜索镜像

docker官网用于查images-镜像的版本

docker images 查看镜像

docker pull 拉取镜像

docker save 导出镜像

docker load 导入镜像

docker tag 打标签

docker image prune 清理镜像

docker inspect 查看镜像信息

  • 使用jq命令对输出的信息进行过滤

docker rmi 删除镜像
