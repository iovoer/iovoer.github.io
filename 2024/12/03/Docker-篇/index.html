docker-容器

容器是隔离的环境中运行的一个进程,如果进程结束,容器就会停止.

容器的隔离环境,拥有自己的ip地址,系统文件,主机名,进程管理,相当于一个mini的系统.

docker安装

本章系统为 Centos-7

docker-images 管理

docker search 搜索镜像

docker官网用于查images-镜像的版本

docker images 查看镜像

-q 返回镜像ID

docker tag 打标签

docker-container 管理

docker restart 重启容器
